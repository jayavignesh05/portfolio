# **App Name**: VisioFolio

## Core Features:

- 3D Parallax Animation: Implement a dynamic 3D parallax scrolling effect for the central image card to engage users as they explore the portfolio.
- Interactive Services Accordion: Create an accordion list of services that expands on click to reveal detailed information, accompanied by a contextually relevant image.
- Horizontally Scrolling Projects: Design a horizontally scrolling showcase of featured projects with full-width images, category tags, titles, and descriptions.
- Highlighted Testimonials: Lay out client testimonials in a grid, highlighting impressive stats using the accent color to draw attention to key achievements.
- Dynamic Contact Form: Develop a contact form with a modern, minimal design, featuring clean fields and a vibrant submission button for potential clients to connect.

## Style Guidelines:

- Background color: Dark textured gray (#333333) to create a premium, dark-mode aesthetic.
- Primary color: Crisp white (#FFFFFF) for text to ensure readability and contrast.
- Accent color: Neon lime green (#39FF14) for highlights, icons, and call-to-action elements, creating a vibrant and modern feel.
- Body and headline font: 'Inter' (sans-serif) to maintain a modern and clean look for all text elements. Note: currently only Google Fonts are supported.
- Custom, circular lime-green cursor that follows the mouse.
- Sticky navigation bar at the top with logo, centered menu links, and a contrasting "Contact" button on the right.
- Implement fluid, interactive animations and transitions throughout the portfolio to enhance user engagement and provide a professional feel.
